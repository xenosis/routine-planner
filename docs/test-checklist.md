# 수동 테스트 체크리스트

변경 후 실기기에서 확인할 시나리오 목록.  
계산 로직 변경 시 → `npm test` 먼저 실행 후 관련 항목 확인.

---

## 일정 탭

- [ ] **한글 입력** — 추가/수정 화면에서 제목·장소·작성자·메모 필드에 한글 입력 시 자모 결합 정상 작동 (ㅎ+ㅏ→하, ㅎ→하→한 등)
- [ ] 일정 추가 (제목, 날짜, 시간, 카테고리, 메모)
- [ ] 일정 수정 (제목 변경, 날짜/시간 변경)
- [ ] 일정 삭제
- [ ] 여러 날 일정 (endDate 있는 경우) — 날짜 범위 bar 표시 확인
- [ ] 이름표 (nameTag) 추가 및 색상 dot 표시
- [ ] 장소 검색 모달 — 카카오 지도 WebView 표시 및 장소 선택
- [ ] 반복 일정 추가 (daily / weekly / monthly / yearly)
- [ ] 반복 일정 수정 / 삭제
- [ ] 반복 종료일(repeatUntil) 설정 및 적용
- [ ] 일정 알람 — 등록, 알림 수신, 탭 시 일정 탭으로 이동
- [ ] 실시간 동기화 — 다른 기기에서 일정 변경 시 자동 반영

---

## 루틴 탭

- [ ] 루틴 추가 (daily / weekly_days / weekly_count)
- [ ] 루틴 수정 (빈도 유형 변경 포함)
- [ ] 루틴 삭제
- [ ] 오늘 체크 / 체크 취소
- [ ] weekly_count quota 달성 시 카드 흐려짐 + 체크 비활성
- [ ] weekly_count quota 달성 후 오늘 목록에서 제외 (오늘 체크한 것은 유지)
- [ ] 스트릭 증가 확인 (daily: 연속 일수, weekly_days: 연속 횟수, weekly_count: 연속 주수)
- [ ] 루틴 알람 — 등록, 알림 수신, 탭 시 루틴 탭으로 이동

---

## 할일 탭

- [ ] 할일 추가 (제목, 마감일, 알람)
- [ ] 할일 완료 처리 → 알람 취소 확인
- [ ] 할일 삭제
- [ ] 마감 후 알람 (마감 +1일 / +1주 / +1달) 수신

---

## 성과 탭

- [ ] 오늘 완료 카운트 정확성 (루틴 탭 완료 수와 일치)
- [ ] 주간 달성률 — 오늘 완료 0회이면 0%
- [ ] 주간 달성률 — weekly_days 루틴 중 이번 주 아직 예정일 없는 것은 제외
- [ ] 주간 차트 — 0%인 날에 "0%" 레이블 표시
- [ ] 주간 차트 — 완료율에 따른 막대 색상 (80% 이상 초록, 50% 이상 인디고)
- [ ] 월간 달력 — 전체 완료(초록) / 일부 완료(노랑) / 미완료(빨강) dot
- [ ] 월간 달력 — 이전 달 이동 및 데이터 로딩
- [ ] 루틴별 달성률 — 완료 횟수 / 기준 횟수 정확성
- [ ] 루틴별 달성률 — weekly_count: 경과 주수 × quota 기준

---

## 알람 (공통)

- [ ] 앱 실행 중 알림 수신 → 해당 탭으로 이동
- [ ] 앱 백그라운드 상태에서 알림 탭 → 해당 탭으로 이동
- [ ] 앱 완전 종료(killed) 후 알림 탭 → 앱 재실행 후 해당 탭으로 이동
- [ ] 반복 일정 알람 — 알림 탭 후 다음 발생 알람 자동 재등록
